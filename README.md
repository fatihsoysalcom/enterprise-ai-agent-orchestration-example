# Enterprise AI Agent Orchestration Example

This example demonstrates the core concept of an 'Enterprise AI Agent' as described in the article. Instead of directly calling a single AI model, the agent orchestrates multiple components (a simulated LLM and a data lookup tool) to achieve a complex goal. It showcases how an agent can plan steps, use tools, and synthesize information for a more intelligent and context-aware AI solution.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
